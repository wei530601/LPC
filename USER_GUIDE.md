# Pi Panel 使用指南

## 主要功能

### 1. 仪表板（系统监控）

实时监控系统状态：

- **CPU 使用率**：显示每个核心的使用情况
- **内存使用**：包括物理内存和 Swap
- **磁盘使用**：显示所有挂载点的使用情况
- **系统温度**：CPU 温度监控
- **网络流量**：实时上传下载速度
- **运行时间**：系统运行时长

仪表板每 2 秒自动刷新一次。

### 2. 服务管理

管理系统 systemd 服务：

- **查看服务列表**：显示所有系统服务
- **启动服务**：启动已停止的服务
- **停止服务**：停止运行中的服务
- **重启服务**：重启服务
- **搜索过滤**：快速查找特定服务

常用服务示例：
- `nginx` - Web 服务器
- `ssh` - SSH 服务
- `cron` - 定时任务
- `docker` - Docker 容器服务

⚠️ **注意**：操作系统服务需要 sudo 权限，请确保运行用户有相应权限。

### 3. Web 终端

功能完整的 bash 终端：

- **交互式命令行**：完整的 bash shell 环境
- **实时输出**：通过 WebSocket 实时显示
- **窗口调整**：自动适应浏览器窗口大小
- **复制粘贴**：支持标准的复制粘贴操作

使用技巧：
- `Ctrl+C`：中断当前命令
- `Ctrl+L`：清屏
- `Tab`：自动补全
- 右键粘贴（部分浏览器）

### 4. 文件管理器

浏览和管理服务器文件：

#### 浏览文件
- 点击文件夹进入
- 点击 ".." 返回上级目录
- 面包屑导航快速跳转

#### 文件操作
- **新建文件夹**：创建新目录
- **新建文件**：创建空文件
- **上传文件**：从本地上传
- **下载文件**：下载到本地
- **编辑文件**：在线文本编辑器
- **删除**：删除文件或空文件夹

⚠️ **权限说明**：
- 默认根目录为 `/home/pi`
- 只能访问配置的根目录及其子目录
- 删除前会弹出确认对话框

## 快捷键

- **Ctrl+/**：（终端）中断命令
- **Ctrl+L**：（终端）清屏
- **Esc**：关闭模态框

## 常见问题

### Q: 为什么看不到系统温度？

A: 温度监控依赖硬件传感器，某些系统或虚拟机可能不支持。树莓派通常可以正常显示。

### Q: 服务管理操作失败？

A: 确保运行 Pi Panel 的用户有 sudo 权限，或者在 sudoers 文件中配置了 systemctl 的无密码权限。

```bash
# 编辑 sudoers
sudo visudo

# 添加（将 pi 替换为实际用户名）
pi ALL=(ALL) NOPASSWD: /bin/systemctl
```

### Q: 终端中文显示乱码？

A: 确保系统已安装中文语言包和 UTF-8 locale。

```bash
sudo apt-get install language-pack-zh-hans
sudo locale-gen zh_CN.UTF-8
```

### Q: 文件上传大小限制？

A: 默认限制 100MB，可在 `config.py` 中修改 `MAX_CONTENT_LENGTH`。

### Q: 如何修改密码？

A: 编辑 `config.py` 文件中的 `DEFAULT_PASSWORD`，然后重启服务。

## 技术支持

如遇到其他问题，请查看：

1. 服务日志：`sudo journalctl -u pi-panel -f`
2. 应用日志：查看终端输出
3. 浏览器控制台：F12 查看 JavaScript 错误

## 安全提醒

- ⚠️ 请勿在公网环境直接暴露，建议使用 VPN 或配置反向代理 + SSL
- ⚠️ 定期更新系统和依赖包
- ⚠️ 使用强密码并定期更换
- ⚠️ 限制文件管理器的根目录范围
